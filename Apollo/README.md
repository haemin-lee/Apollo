# Apollo

This is the React Native code.

### Prerequisites

-   React Native
-   XCode

### Running

Install the node modules

```
$ npm i
```

Install the Pods

```
$ npx pod-install ios
```

**Simulator**

Run the app

```
$ npx react-native run-ios
```

**Device**

Open the XCode project under `ios/Apollo.xcworkspace`

Click the Run button
